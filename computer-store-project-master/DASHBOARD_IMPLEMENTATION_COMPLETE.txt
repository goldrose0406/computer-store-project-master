     SELECT COUNT(DISTINCT customerEmail) as count FROM orders

       strftime('%b', createdAt) as month,

     FROM orders
     WHERE status IN ("confirmed", "shipped", "delivered")
     GROUP BY strftime('%m', createdAt)

     SELECT status, COUNT(*) as count FROM orders GROUP BY status

✅ FRONTEND CHANGES - frontend/src/components/DashboardReports.js
═════════════════════════════════════════════════════════════════

THAY ĐỔI:

1️⃣ Import Changes:
   ✅ Added: import { useAuth } from '../context/AuthContext'
   ❌ Removed: Mock generateMockDashboardData()

2️⃣ State Management:
   ✅ Added: const { token } = useAuth()
   ✅ Added: const [revenueData, setRevenueData] = useState([])
   ✅ Added: const [ordersStatsData, setOrdersStatsData] = useState(null)

3️⃣ Fetch Method - TRƯỚC (Mock):
   ❌ const fetchDashboardData = async () => {
        const mockData = generateMockDashboardData();
        setData(mockData);
      };

   SAU (Real API):
   ✅ const fetchDashboardData = async () => {
        // Fetch dashboard - REAL
        const dashboardResponse = await fetch(
          '${API_URL}/reports/dashboard',
          { headers: { 'Authorization': `Bearer ${token}` } }
        );
        const dashboardData = await dashboardResponse.json();

        // Fetch revenue - REAL
        const revenueResponse = await fetch(
          '${API_URL}/reports/revenue?days=30',
          { headers: { 'Authorization': `Bearer ${token}` } }
        );
        const revenueDataFromAPI = await revenueResponse.json();
        setRevenueData(revenueDataFromAPI);

        // Fetch orders stats - REAL
        const ordersResponse = await fetch(
          '${API_URL}/reports/orders',
          { headers: { 'Authorization': `Bearer ${token}` } }
        );
        const ordersStatsFromAPI = await ordersResponse.json();
        setOrdersStatsData(ordersStatsFromAPI);
      };

4️⃣ Data Source - Charts:
   ❌ TRƯỚC: <LineChart data={data.dailyRevenue}>
   ✅ SAU:   <LineChart data={revenueData}>

5️⃣ Table Columns - Top Products:
   ❌ Rating column removed (không có trong real data)
   ✅ Chỉ còn: Product name, Sales quantity, Revenue
